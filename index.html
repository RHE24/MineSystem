MineSystem: An Open Source CMS for Minecraft Server Websites

MineSystem is an open source content management system, designed specifically for Minecraft server owners and administrators. MineSystem is written primarily in PHP, using HTML and CSS to cater for all your website needs. We understand that every good server needs a website - with unlimited flexibility, MineSystem will create that attractive looking website overnight, no programming knowledge required!

Features

MineSystem is designed to be flexible and able to cater for your needs, however great. In addition to the main features listed below, you can find more on our wiki, on GitHub. As an open source project, we encourage you to make your own and share them with the rest of the community.

  • Dynamic Front Page The front page of your website is what visitors will see first - it creates the first impression. It not only needs to look attractive, but also be functional in an easy to use way. The front page can have custom text and HTML, and widgets (more below) can be used to create a super dynamic front page.
  • Customizable Widgets MineSystem would not exist without widgets. Widgets are what create the functionality behind a MineSystem site. MineSystem comes with default widgets, such as login forms, notices and polls, but to really make your server website unique, you can find extra plugins (approved, of course) from the community on our wiki.
  • Fast and Reliable Users System To communicate effectively and privately with your server users, MineSystem has a built in users system. MineSystem provides the functionality of automatic whitelisting from registration and/or console management from the website. Administrators can create user only pages and features, and registered users can post in the forums among other things.

Installation

A more in-depth and detailed guide can be found on our wiki on GitHub. To be able to use MineSystem to its full potential, you need only a couple of things:

1. A Minecraft Server

This server needs to be accessible from any computer with a connection to the internet. In other words, no VPN or LogMeIn Hamachi servers!

2. A Web Server

This web server needs to be running the latest version of PHP and MySQL. For an easy to use web server for Windows, check out EasyPHP.
